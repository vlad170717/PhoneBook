Triples Open on Hyperskill

Write a program that reads an array of ints and outputs the number of "triples" in the array.

A "triple" is three consecutive ints in increasing order differing by 1 (i.e. 3,4,5 is a triple, but 5,4,3 and 2,4,6 are not).

The first line contains the size of an array.
The second line contains elements of the array separated by spaces.

Output only a single integer value - the number of "triples".

In the example below there are two triples: 4,5,6 and 5,6,7.

Sample Input:
6
1 2 4 5 6 7

Sample Output:
2


Memory limit: 256 Mb
Time limit: 5s
